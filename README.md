# POS SaaS 시스템

다양한 업종의 소매점에서 사용 가능한 클라우드 기반 POS(Point of Sale) 서비스.

## 주요 기능

- **회원가입/로그인** — 아이디+비밀번호 (가입 시 전화번호 SMS 인증)
- **다점포 운영** — 1계정으로 여러 가게 관리
- **직원 관리** — 사장이 직원 계정 생성 + 권한 설정
- **업종별 프리셋** — 10개 업종 카테고리 자동 생성
- **판매/결제** — 바코드 스캔, 상품 검색, 현금/카드/혼합 결제
- **도매가 자동 적용** — 일정 수량 이상 시 도매 단가 전환
- **부가세 처리** — 공급가/부가세/합계 자동 분리
- **매출 통계** — 일별/월별 매출, 인기 상품, 카테고리별 매출
- **카드 단말기 연동** — VAN사(NICE/KIS/KSNET) 로컬 에이전트 통신
- **거래처 관리** — 거래처 등록/수정/삭제
- **거래명세서** — 판매 직후 또는 이력에서 발행, 인쇄
- **현금영수증** — 국세청 API 연동, 현금 결제 시 발급
- **일마감/시재 관리** — 시재 입력 → 매출 집계 → 과부족 확인
- **활동 로그** — 환불, 가격 변경 등 주요 활동 감사 추적
- **영수증** — 사업자등록번호 + VAT 표시, 브라우저 인쇄
- **오프라인 모드** — PWA 지원, 인터넷 끊겨도 판매 가능

## 기술 스택

| 구분 | 사용 기술 |
|------|-----------|
| Backend | Django, Django REST Framework, PostgreSQL |
| Frontend | Next.js (React), TypeScript, Tailwind CSS |
| Auth | JWT + 아이디/비밀번호 + SMS 인증(가입 시) |
| Hosting | Railway/Render (백엔드), Vercel (프론트엔드) |
| Monitoring | Sentry (에러 모니터링) |
| CI/CD | GitHub Actions (자동 테스트 + 자동 배포) |

## 지원 업종

일반 소매점, 카페/음료, 음식점, 편의점/마트, 베이커리/제과, 철물점, 의류/패션, 약국/건강, 문구/사무, 직접 설정
