# 대왕철물 POS 시스템

철물점 전용 웹 기반 POS(Point of Sale) 시스템입니다.

인터넷 없이 오프라인 환경에서도 동작하며, 상품 관리부터 판매, 결제, 매출 통계까지 철물점 운영에 필요한 핵심 기능을 제공합니다.

## 주요 기능

- **판매/결제** — 바코드 스캔, 상품 검색, 장바구니, 현금/카드/혼합 결제, 거스름돈 자동 계산
- **도매가 자동 적용** — 일정 수량 이상 구매 시 도매 단가 자동 전환
- **상품 관리** — 등록/수정/삭제, 카테고리 분류, 소수점 수량(m/kg) 지원
- **판매 이력** — 날짜별 조회, 거래 상세, 환불 처리
- **영수증 인쇄** — 80mm 열전사 프린터 지원
- **매출 통계** — 일별/월별 매출, 인기 상품 TOP 10, 카테고리별 매출, 결제 방법 비율 (Chart.js)
- **재고 관리** — 재고 부족 알림, 판매 시 자동 차감
- **DB 백업** — 서버 시작 시 자동 백업 + 수동 백업

## 기술 스택

| 구분 | 사용 기술 |
|------|-----------|
| Backend | Python, Flask, SQLite |
| Frontend | Jinja2, Bootstrap 5, Chart.js |
| 특징 | CDN 의존 없음 (완전 오프라인), ORM 없이 순수 SQL |

## 실행 방법

```bash
pip install flask
python pos.py
```

브라우저에서 http://localhost:5000 접속

### 샘플 데이터 등록 (선택)

```bash
python setup_sample_data.py
```

철물점 샘플 상품 50개가 자동 등록됩니다.

## 프로젝트 구조

```
pos.py              ← Flask 앱 + 모든 라우트
config.py           ← 설정 (가게명, 단위 유형, 카테고리)
database.py         ← SQLite 스키마, DB 연결/백업
models.py           ← 데이터 액세스 함수 (순수 SQL)
templates/          ← Jinja2 HTML 템플릿 (9개)
static/             ← CSS, JS, 폰트 (로컬 번들)
setup_sample_data.py ← 테스트용 샘플 데이터
```
